Project Ideas

  • Overview
    • I want to create a website that belong to "nutrition" category named "Nutrient Calculator". Nutrient Calculator has one main feature which is to calculate the nutrients like calorie, protein, fats, carbohydrates your body need in one day,week or month when given body's statistic such as weight and height. It also has a lot more interesting features within.
  • Motivation
    • I believe "Nutrient Caculator" is a really interesting and useful website that support many needs of people. Since the beginning, healthcare is always important no matter what situation, this is why hospitals, medicines are vital. However, if those help you when you are already sick, take care of your body so it doesn't get sick is as important. For your body to be well, you don't need a big amount of nutrition, you need a "right" amount of nutrition, and Nutrient Calculator can help you with that.
  • Description
    • Nutrient Calculator has 4 features:
    1. It will ask for your body's status, then it will begin to calculate the amount of nutrient (protein,calorie,fat, etc) base on your choice that is suitable for your body to absorb. You can choose between three options, which are per day, per week and per month, it will divide or multiply accordingly.
    2. (Ideally) The website is connect to a database which has a list of food ingredients, each one has a value of nutrition per 100 gram. Users can use this "wiki" to search for information they want to know.
    3. (Ideally) You can choose different ingredients as well as well their amount (in term of gram) and add up to a Calculator. This calculator will give you the serving size of nutrition for each meal. By doing this users can control the nutrients they eat better per day.
    4. (Ideally) Before calculating the amount of nutrient, there are many check boxes that are labled with different diseases and condition such as diabetes, high/low blood pressure, etc . When checked, the final amount of nutrient your body need will raise up/lower down some percentage accordingly.
  • Tools and Technologies
    • Physical tools: a PC or laptop, pen and paper to do sketches/prototype.
    Software: Visual Studio Code for coding both front end and back end. MySQL and Pycharm for database.
    Coding Languages: HTML, CSS, JavaScript and PHP.
  • Skills Required
    • 1. Know how to use software like VSCode, MySQL, Pycharm and the Coding Languages that are associated with them (HTML, CSS, JavaScript, PHP)
    2. Ability to work as a team through communication and the use of git,github.
    These requirements are not hard to acquire. Everyone have a PC or laptop. All the software listed above are free, the coding Languages can be learned through guide online such as https://www.w3schools.com. However, you can do it is different than you can do it "well". To be able to make a good website, i need to try my best.
  • Outcome
    • If Nutrient Calculator is finished ideally, all kinds of people are more or less benefits. Whether you are an gymnastic who want to control your nutrient, an patience who have a carbohydrates problem, or just someone who interest on seeing the required nutrients, you may want to use this app. As for the impact, this project isn't about some great features that will shake the world, but i believe it can help everyone, if they choose to use it.
